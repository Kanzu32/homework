Групповой преобразование делигируемых методов.

Групповое преобразование позволяет присвоить имя метода делегару не прибегая к оператору new или явному вызову конструктора делегата.
(>= C№ 2.0)

Автоматрическое преобразование метода в тип делегата осуществяется средствами С№.

StrMod StrOp = new StrMod(...);
StrMod StrOp = ...;

Ковариантнасть и контравариантность

Ковариантнасть позволяет присвоить делегату метод возвращаемым типом которого служит класс производный от класса
указываемым в указаном возвращаемом типе делегата.

Контравариантность позволяет присвоить метод типом которого является класс который являющийся базовым
для класса указываемым в обьявлении делигата

Разновидности анонимных функций (начиная с С№ 3.0)

+ анонимные методы
+ лямбда выражения

Анонимная функция представляет собой безымянный кодовый блок предоставляемый делегату.

для создания анонимного метода необходимо указать кодовый блок после ключевого слова delegate { };

Локальная переменная в область действия которой входит анонимный метод называется внешней переменной.
В случае её использования в анонимном методе она называется захваченной.
захваченная переменная существует до тех пор пока захвативший её делегат не будет уничтожен сборщиком мусора.

События

Событе является автоматическим уведомлением о том что произошло некоторое действие

Принцип действия
объект, проявляющий интерес к собвтию, регистрирует обработчик этого события. Когда событие происходит вызываются все зарегестрированные
обработчики события.

В случае если статическиц метод используется как обработчик уведомление о событии распространяется на весь класс.
В случае если в качестве обработчика используется метод экземпляра, то события адресуются конкретным экземпляром объектов,
то события адресуется конкретным экземпляром объектов ...
